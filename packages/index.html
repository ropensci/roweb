rOpenSci packages

This is a complete list of all available rOpenSci packages. Packages are grouped by one that acquire data, full-text of journal articles, altmetrics, data-publication and data visualization. Packages with a cran sign are stable versions that you can instantly install from your local mirror using install.packages("PACKAGE_NAME"). Others are in various stages of development (bleeding edge packages are not listed here) and you can learn more by following our GitHub organization page.
